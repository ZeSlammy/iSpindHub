Settings for iSpindHub

Use the tabs above to navigate the various settings and customize iSpindHub for your own use. Be sure to use the update button after changing a field.

Controller Settings

URL Target Settings

  • These settings control how Brew iSPindHub will relay iSpindel data to an HTML endpoint such as BrewPi Remix or Fermentrack. To disable pushing to the endpoint, delete the URL.

Brewer's Friend Target Settings

  • These settings control how iSpindHub forwards iSpindel information to Brewer's Friend. With a premium account, Brewer's Friend supports logging brew-related data. To find your API key; log into your Brewer's Friend account and go to Profile > Account. Your API key will be a 20 to 64-character string of letters and numbers (not a URL).

    To disable pushing to Brewer's Friend, delete the key.

Brewfather Target Settings

  • These settings control how iSpindHub forwards iSpindel information to Brewfather. With a premium account, Brewfather supports logging brew-related data. To find your API key; log into your Brewfather account and go to Custom Stream. Your API key will be a 10 to 64-character string of letters and numbers on the line following "URL http://log.brewfather.net/stream?id=" (do not include the the URL.)

    To disable pushing to Brewfather, delete the key.

Reset Controller

You may want to reset the controller for some reason. For instance if it's AAALLL the way in the garage. You can do that here. This is a safe operation, it will simply reboot the controller.
